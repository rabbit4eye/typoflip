TypeFlip — Web Prototype

This is a working prototype of the TypeFlip concept — a macOS utility that instantly fixes text typed in the wrong keyboard language.

Problem: Bilingual typists (Thai/English) experience 10–30 wrong-language incidents per day. Each costs ~5–10 seconds of delete-switch-retype, totaling 15–20 hours per year — plus constant mental irritation.

Solution: Highlight → Hotkey → Instant conversion. The engine maps between English QWERTY and Thai Kedmanee layouts with O(n) character mapping, enhanced by character-frequency language detection.
